This calculator determines an adjusted tariff rate based on GDP per capita, Purchasing Power Index gap (PPI Gap), and existing tariff offset. It uses a relative development gap against the U.S. and a weighting factor delta (δ) to scale fairness adjustments. No raw tariff input is used; all results are derived from contextual data.
